This security dashboard uses the OWASP ZAP (Zed Attack Proxy) to automatically scan your Zomato application for security vulnerabilities. The scanner identifies issues like:

  • SQL Injection: Attackers can manipulate your database queries
  • Cross-Site Scripting (XSS): Malicious scripts injected into web pages
  • Authentication Weaknesses: Problems with login systems
  • Information Disclosure: Sensitive data accidentally exposed
  • And many more security vulnerabilities...

The system will categorize issues by risk level and provide detailed information on how to fix them.
